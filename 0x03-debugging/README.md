# Debugging
In this project, we were tasked with solving some debugging tasks.

## Tasks

1. [0-main.c](./0-main.c): Prints if an integer is positive or negative

2. [1-main.c](./1-main.c): Causes an infinite loop

3. [2-largest_number.c](./2-largest_number.c): Returns the largest number of three given as input

4. [3-convert_day.c](./3-convert_day.c): Converts day of the month to days of the year

5. [3-print_remaining_days.c](./3-print_remaining_days.c): Calculates the remaining days of the year from a given date
